## 🚀 How to Use? / Nasıl Kullanılır?

### [TR]
Bilgisayarınızda hiçbir programlama dili veya ek kütüphane kurulu olmak zorunda değil! 
1. Depodaki **`PEVEPERO_Ultimate_Booster.bat`** dosyasını indirin.
2. Dosyaya sağ tıklayın ve **"Yönetici Olarak Çalıştır"** seçeneğini seçin.
3. Açılan renkli terminalde dilinizi seçin (`1` Türkçe).
4. İşlem bittiğinde, değişikliklerin aktif olması için **bilgisayarınızı yeniden başlatın!**

### [EN]
No programming languages or additional libraries are required on your system!
1. Download the **`PEVEPERO_Ultimate_Booster.bat`** file from this repository.
2. Right-click the file and select **"Run as Administrator"**.
3. Choose your language in the colorful terminal interface (`2` English).
4. Once completed, **restart your PC** for the deep kernel modifications to take effect!
