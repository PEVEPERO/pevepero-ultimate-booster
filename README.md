# PEVEPERO Ultimate Booster v1 🚀

An open-source, dual-language (TR/EN) universal kernel and hardware optimization script designed to maximize system performance after a fresh Windows format.

## 🛠️ Features / Özellikler

### English (EN)
- **Ultimate Power Scheme:** Forces Windows into hidden ultimate performance mode and unlocks CPU cores.
- **HPET De-synchronization:** Eliminates micro-stutters in high-FPS games.
- **Cloudflare Gaming DNS:** Automatically injects optimized low-latency DNS.
- **Advanced Hardware Control:** Disables Windows PowerThrottling at a kernel level.

### Türkçe (TR)
- **Nihai Güç Planı:** Gizli nihai performans modunu otomatik aktifleştirir ve işlemci parklarını kaldırır.
- **HPET Zamanlayıcı Yaması:** Oyunlardaki anlık mikro takılmaları (stutter) yok eder.
- **Cloudflare Gaming DNS:** Gecikme süresini düşürmek için sistem ağ kartlarına düşük pingli DNS enjekte eder.
- **Gelişmiş Donanım Kontrolü:** Kernel seviyesinde güç kısılmalarını (Throttling) engeller.

## 🚀 How to Use / Nasıl Kullanılır?

1. Download the `PEVEPERO_Ultimate_Booster.bat` file.
2. Right-click on the file and select **"Run as Administrator"**.
3. Choose your language (1 for TR, 2 for EN) and follow the on-screen instructions.
